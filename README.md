# 是男人就下100层

一个经典的平台跳跃游戏，使用纯HTML5 Canvas和JavaScript开发。

## 游戏说明

控制角色不断向下跳跃，尽可能到达更深的层数。避开红色危险平台，跳到绿色平台可以恢复生命值。

### 操作方式

- **← →** 或 **A D** 键：左右移动
- **空格键** 或 **↑** 或 **W** 键：跳跃

### 游戏规则

- 玩家需要不断向下跳跃
- 红色平台会减少生命值
- 绿色平台可以恢复生命值
- 生命值归零或掉出屏幕游戏结束
- 挑战到达100层！
